Animated Email Signature

Hyperlinked, animated (GIF-based, so it plays in Gmail, Apple Mail and Outlook web). Logo swipes in left to right, the photo assembles from diagonal slices, and the icons pop in one by one.

⚠️ You're previewing from a local file — the copied signature references images by absolute URL, so deploy this folder first (it's part of the site: npx wrangler deploy), then copy from the deployed page so the GIF URLs resolve for your recipients.

Install in Gmail

  1. Click Copy signature above.
  2. Gmail → ⚙️ → See all settings → General → Signature → Create new.
  3. Paste (Ctrl/Cmd+V) into the signature box and save. The GIFs animate automatically.

Install in Apple Mail

  1. Copy the signature, then Mail → Settings → Signatures → + and paste.
  2. Untick “Always match my default message font”.

Install in Outlook (web / new Outlook)

  1. Settings → Account → Signatures → paste. Note: classic desktop Outlook (Word engine) shows only the first GIF frame — the layout and links still work, it just displays the finished, fully-revealed frame.

Customize

  1. Name, title, links: edit the table in signature/index.html.
  2. Photo, logo text, icons, animation timing: edit signature/generator/render.html, then run node signature/generator/build.mjs to regenerate the GIFs (to use your own portrait, point the .ph background-image at your photo).
